# Corvus Display

Brightness, color temperature, and resolution control for external and non-Apple displays on Apple Silicon Macs. Hardware DDC where it is available, smooth software dimming everywhere else, all from your menu bar.

**[Download and learn more →](https://corvusdevs.github.io/CorvusDisplay/)**

- One-time purchase, 14-day free trial, no account.
- Per-app profiles, presets, keep-awake, sleep timer, multi-display.
- Private by design: no telemetry, no tracking, runs entirely on your Mac.

Requires macOS 14 Sonoma or later, Apple Silicon.

This repository hosts the public website (GitHub Pages) and signed release downloads. The app source lives in a private mirror.
